# Getting Started

[Geting started](https://vuejs.org/v2/guide/)

Install CLI tools globally
`npm install -g @vue/cli`

Create a project
`vue create spa`
default v2 project

This generates a README with instructions on build and run
`npm install`
`npm run serve` to launch dev live server
